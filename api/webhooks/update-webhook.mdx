---
title: "Update webhook"
description: "Update endpoint URL or enabled status."
api: "PATCH /webhooks/{endpoint_id}"
---

Updates an existing endpoint owned by the authenticated user.

## Path parameters

<ParamField path="endpoint_id" type="string" required>
  Webhook endpoint UUID.
</ParamField>

## Request body

<ParamField body="url" type="string">
  New destination URL.
</ParamField>

<ParamField body="enabled" type="boolean">
  Enable or disable the endpoint.
</ParamField>

At least one field should be provided.

## Response

Returns the updated endpoint object.

## Example

<CodeGroup>

```bash cURL
curl -X PATCH "https://api.proxyjam.com/public/v1/webhooks/wh_01HXYZ" \
  -H "X-API-Key: pj_AbCdEf..." \
  -H "Content-Type: application/json" \
  -d '{"enabled": false}'
```

```python Python
from proxyjam import ProxyJamClient

with ProxyJamClient.from_env() as client:
    updated = client.webhooks.update("wh_01HXYZ", enabled=False)
    print(updated.enabled, updated.url)
```
