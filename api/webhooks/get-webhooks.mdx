---
title: "List webhooks"
description: "Get webhook endpoints configured for the authenticated user."
api: "GET /webhooks"
---

Returns webhook endpoints for the current user. For now, only `proxy.status_changed` is supported.

## Query parameters

<ParamField query="limit" type="integer">
  Number of items to return. Default is `50`, max is `100`.
</ParamField>

<ParamField query="offset" type="integer">
  Number of items to skip before collecting the result set.
</ParamField>

## Response

<ResponseField name="items" type="array">
  List of webhook endpoint objects.
</ResponseField>

Each endpoint includes:

- `id`
- `event_type`
- `url`
- `enabled`
- `secret_preview`
- `last_test_at`
- `last_delivery_at`
- `created_at`
- `updated_at`

## Example

<CodeGroup>

```bash cURL
curl https://api.proxyjam.com/public/v1/webhooks \
  -H "X-API-Key: pj_AbCdEf..."
```

```python Python
from proxyjam import ProxyJamClient

with ProxyJamClient.from_env() as client:
    endpoints = client.webhooks.list()
    for hook in endpoints.items:
        print(hook.id, hook.url, hook.enabled)
```
