---
title: "Send test webhook"
description: "Send one immediate test delivery to an endpoint."
api: "POST /webhooks/{endpoint_id}/test"
---

Triggers a signed test delivery for the selected endpoint. Test deliveries are persisted for diagnostics but are **not retried**.

## Path parameters

<ParamField path="endpoint_id" type="string" required>
  Webhook endpoint UUID.
</ParamField>

## Response

Returns:

- endpoint details
- delivery state (`delivered`, `failed`, `blocked`, etc.)
- attempt timeline including response status and latency

## Signature headers

Each request contains:

- `X-ProxyJam-Event`
- `X-ProxyJam-Delivery-Id`
- `X-ProxyJam-Timestamp`
- `X-ProxyJam-Signature`

## Example

<CodeGroup>

```bash cURL
curl -X POST "https://api.proxyjam.com/public/v1/webhooks/wh_01HXYZ/test" \
  -H "X-API-Key: pj_AbCdEf..."
```

```python Python
from proxyjam import ProxyJamClient

with ProxyJamClient.from_env() as client:
    result = client.webhooks.send_test("wh_01HXYZ")
    print(result.delivery.delivery.status, result.delivery.delivery.last_http_status)
    for attempt in result.delivery.attempts:
        print(attempt.attempt_no, attempt.response_status, attempt.latency_ms)
```
