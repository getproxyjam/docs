---
title: "Get notification preferences"
description: "Return the user's notification settings for each channel."
api: "GET /notifications/preferences"
---

Returns whether the user has enabled notifications for each delivery channel.

## Response

<ResponseField name="preferences" type="array">
  <Expandable title="NotificationPreferenceResponse">
    <ResponseField name="id" type="string">Preference UUID.</ResponseField>
    <ResponseField name="user_id" type="string">Owner's user UUID.</ResponseField>
    <ResponseField name="channel" type="string">Channel: `in_app` or `email`.</ResponseField>
    <ResponseField name="enabled" type="boolean">Whether the channel is enabled.</ResponseField>
    <ResponseField name="created_at" type="string">ISO 8601 timestamp.</ResponseField>
    <ResponseField name="updated_at" type="string">ISO 8601 timestamp of last change.</ResponseField>
  </Expandable>
</ResponseField>

## Example

<CodeGroup>

```bash Request
curl https://api.proxyjam.com/public/v1/notifications/preferences \
  -H "Authorization: Bearer <access_token>"
```

```json Response
{
  "preferences": [
    {
      "id": "pref-uuid-1",
      "channel": "in_app",
      "enabled": true,
      "created_at": "2024-01-15T10:00:00Z",
      "updated_at": "2024-01-15T10:00:00Z"
    },
    {
      "id": "pref-uuid-2",
      "channel": "email",
      "enabled": true,
      "created_at": "2024-01-15T10:00:00Z",
      "updated_at": "2024-01-15T10:00:00Z"
    }
  ]
}
```

</CodeGroup>
