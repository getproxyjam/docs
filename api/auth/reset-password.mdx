---
title: "Reset password"
description: "Set a new password using a reset token."
api: "POST /users/reset-password"
---

Sets a new password for the account associated with the reset token. The token is obtained from the password reset email.

## Request body

<ParamField body="token" type="string" required>
  Password reset token from the email.
</ParamField>

<ParamField body="new_password" type="string" required>
  The new password.
</ParamField>

<ParamField body="confirm_password" type="string" required>
  Must match `new_password`.
</ParamField>

## Response

<ResponseField name="message" type="string">
  `"Password reset successfully"`
</ResponseField>

## Example

```bash
curl -X POST https://api.proxyjam.com/public/v1/users/reset-password \
  -H "Content-Type: application/json" \
  -d '{
    "token": "abc123...",
    "new_password": "<new-password>",
    "confirm_password": "<new-password>"
  }'
```
