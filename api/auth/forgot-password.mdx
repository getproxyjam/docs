---
title: "Forgot password"
description: "Request a password reset email."
api: "POST /users/forgot-password"
---

Sends a password reset link to the provided email address.

<Note>
  The response is always `200 OK` regardless of whether the email belongs to an existing account. This prevents user enumeration.
</Note>

## Request body

<ParamField body="email" type="string" required>
  Email address of the account to reset.
</ParamField>

## Response

<ResponseField name="message" type="string">
  `"Password reset instructions sent if account exists"`
</ResponseField>

## Example

```bash
curl -X POST https://api.proxyjam.com/public/v1/users/forgot-password \
  -H "Content-Type: application/json" \
  -d '{"email": "you@example.com"}'
```
